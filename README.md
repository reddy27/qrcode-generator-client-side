#Client side QRcode generator.

#Getting started

1. Go to project folder and install dependencies:
 ```bash
 npm install
 ```

2. Run 
  ```bash
 npm start:dev
 ```

#How to build
Run 
  ```bash
 npm build
 ```

#To generate new QRCode on 'src/index.js' file replace 'you rock' string with the QRCode text which you would like.
